A alguns meses atrás assisti uma palestra no Youtube do Zeno Rocha sobre, Web Components.

O Zeno, falava do que estava ocorrendo na web naquele momento e tentava prever o futuro. Web componentes é uma especificação da w3c lançada no meiado de 2013, que nos permiti encapsular códido para criação de template em forma de novas tags e/ou atributos.

A Google e Mozilla, se aderiram a novidade e criaram sua biblioteca para nos ajudar nest tarefa e criarmos nossos experimentos. A Mozilla desenvolveu a X-Tag e a Google o Polymer.

O Polymer parece estar mais avançado e foi este que adotei para criar o web component "".

O que é o elemento <web-loader>?

É um simples web component, que exibi alguns diferentes loaders. Veja o exemplo: http://ifly9.com.br/webcomponents/web-loader/.

Hoje foi apenas a primeira etapa de estudos, pretendo expandir este componente e desenvolver outros!
